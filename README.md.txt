# Chess Simulator Engine

A fully-featured chess interactive chess analyzer built with Python and Pygame, featuring:
- Full chess gameplay with legal move validation
- Pawn promotion with piece selection dialog
- Stockfish engine integration for move analysis
- Position evaluation bar

- Python 3.7+
- Pygame
- python-chess
- Stockfish engine
